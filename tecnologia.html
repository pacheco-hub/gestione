la nostra tecnologia

Le basi del nostro sviluppo software

PHP

Acronimo di Personal Home Page, è un linguaggio di programmazione tanto rivoluzionario quanto semplice e gode di grande popolarità, secondo gli ultimi dati riportati da w3techs, PHP è infatti utilizzato dal 78% dei siti, tra cui i noti Facebook, WordPress e Wikipedia. E' sicuro, affidabile, veloce e altamente prestante

Javascript

Javascript è un linguaggio di programmazione orientato agli oggetti e agli eventi, comunemente utilizzato nella programmazione Web lato client (esteso poi anche al lato server) per la creazione, in siti web e applicazioni web, di effetti dinamici interattivi tramite funzioni di script invocate da eventi innescati a loro volta in vari modi dall'utente sulla pagina web in uso (mouse, tastiera, caricamento della pagina ecc...).

MySQL

MySQL è un relational database management system (RDBMS) composto da un client a riga di comando e un server, entrambi disponibili sia per sistemi Unix e Unix-like sia per Windows; le piattaforme principali di riferimento sono Linux

Bootstrap

Gli applicativi utilizzati sono bootstrap, il quale è una raccolta di strumenti liberi per la creazione di siti e applicazioni per il Web. Esso contiene modelli di progettazione basati su HTML e CSS, sia per la tipografia, che per le varie componenti dell'interfaccia, come moduli, pulsanti e navigazione, così come alcune estensioni opzionali di JavaScript.

Server e tecnologia utilizzata

Il sistema operativo del server è AlmaLinux 8.8 (Sapphire Caracal), è una distribuzione linux per server 64bit.

Il database è MySql

Il server è una macchina di proprietà Andilab presente nella farm di ARUBA SPA

Sicurezza

L’accesso ai servizi avviene tramite spid per garantire il più alto livello di sicurezza

La navigazione è in protocollo https e permette la criptografia dei dati

Antivirus

Il sistema dispone di un antivirus Linux denominato Antivirus Plesk Premium, il quale verifica i file caricati, le possibili minacce, ed è sempre aggiornato.

Spazio di archiviazione

Lo spazio dedicato per l’intera soluzione comprende file della soluzione Andilab fornita all’istituto.

L'istituto può aumentare lo spazio di archiviazione.

I testi, le immagini e i documenti caricati saranno presenti nel server e richiamabili in ogni momento.

L’archivio è accessibile da parte dell’utente ogni qual volta lo desidera.
